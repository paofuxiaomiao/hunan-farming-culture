# 湖南省农耕文化地图 · 设计规范

## 设计哲学：山水咨询主义（Landscape Consulting）

McKinsey式咨询版式骨架 × 湖湘山水雾面自然气质。核心信条：**留白即结构、自然即隐喻、网格即纪律、一页一论点**。

本项目不局限于传统政府网站的呆板布局，而是以高端文化展示为定位，融合自然美学与现代交互技术，打造一个有呼吸感、有格调、有沉浸体验的农耕文明数字展厅。

---

## 色彩系统

全篇遵循「自然 90% + 强调 10%」的配色比例。大面积为暖雾白与深松墨绿的自然基调，土金色作为品牌强调色（替代原skill中的紫色），出现即意味着「农耕文化价值点」。

| 令牌 | 色值 | 用途 |
|---|---|---|
| 暖雾白 `--background` | oklch(0.983 0.003 90) | 全局底色，纸感呼吸 |
| 深松墨绿 `--ink` | oklch(0.26 0.03 168) | 主文字色，山的重量 |
| 土金色 `--brand` | oklch(0.52 0.12 75) | 唯一高饱和强调色：数字、关键词、品牌标识 |
| 雾青 `--mist` | oklch(0.72 0.03 178) | 次要点缀、图表辅助色 |
| 浅石灰 `--stone` | oklch(0.92 0.006 96) | 卡片描边、分区底色 |
| 古代标注 | oklch(0.52 0.12 75) | 古代农耕遗址（棕金） |
| 现代标注 | oklch(0.52 0.12 155) | 现代农耕地标（绿） |
| 红色标注 | oklch(0.52 0.18 25) | 红色农耕旧址（红） |

**禁则**：品牌金色绝不做大面积背景或渐变装饰；不使用高饱和撞色；暗色页使用摄影图自带的深色，不另造纯黑底。

## 版式系统

**非中心化左轴布局**为主。签名元素：
- 等高线纹理背景（`.contour-bg`，透明度 <6%）
- 细分隔线（`.hairline`，左深右浅渐隐）
- 品牌粒子（`.brand-dot`，7px 金色小方块）
- 每页 ≥40% 负空间

---

## 动效系统

| 动效 | 参数 |
|---|---|
| 页面切换 | 650ms「雾散开」opacity + translateY + blur |
| 元素入场 | 900ms reveal，每元素延迟 30–80ms |
| 全幅摄影 | 18s Ken Burns scale(1.06→1) |
| 3D视差 | mousemove驱动 rotateX/Y ≤4° |

缓动：`--ease-out-quint: cubic-bezier(0.23,1,0.32,1)`

**禁则**：禁止弹跳、旋转、彩色闪烁；禁止 scale(0) 起始。

## 页面结构

### 首页（地图浏览）
- 全屏Hero区：湖南山水摄影 + 大标题 + 数据统计
- 地图区：Google Maps + 三种图层 + 左侧筛选 + 右侧详情
- 底部快速导航到各子页面

### 主题路线
- 全幅摄影Hero
- 三条路线卡片，每条配大图 + 路线点位 + 描述

### 发展脉络
- 横向时间轴，滚动驱动
- 每个时期配图 + 文字 + 地点

### 二十四节气（重点页面）
- **使用 Three.js / WebGL 粒子系统**
- 背景不是纯色，而是动态粒子/雾气/自然纹理
- 植物画册插画作为主视觉，带3D视差
- 滚轮/键盘/点击翻页
- 每个节气有独特的粒子颜色和动态背景

### 重要文物
- 网格画廊式布局
- 悬浮展开详情
- 图片放大查看

## 24节气交互技术方案

使用 `three.js` 或 `@react-three/fiber` 创建：
1. **粒子背景**：根据季节变化粒子颜色和运动方式（春：花瓣飘落；夏：萤火虫；秋：落叶；冬：雪花）
2. **雾面纹理**：使用 shader 创建动态雾气效果
3. **植物画视差**：鼠标驱动的3D倾斜 + 景深模糊
4. **过渡动画**：节气切换时粒子重组动画

备选方案（如Three.js过重）：
- 使用 `tsparticles` 轻量粒子库
- CSS 3D transforms + 自定义 canvas 粒子
- Lottie 动画配合 framer-motion

## Brand Voice

标题使用结论式写法，克制陈述。禁止「欢迎访问」「点击了解更多」等套话。

示例：
- ✅「万年稻作，始于湖湘」
- ✅「一粒稻种，改变世界的起点」
- ❌「欢迎来到湖南省农耕文化地图」
